# Mock Database Structure

This folder contains JSON files that represent the initial data of the "Gestão KZ" system.
During development, the application uses `localStorage` for real-time persistence.

To "save" your data locally:
1. Go to the Home screen in the application.
2. Click on **"Exportar Backup"**.
3. Replace the files in this folder with the exported JSON to keep your mock data up-to-date.

This structure is ready to be replaced by a real database (e.g., PostgreSQL, Firebase, or MongoDB) when you integrate a backend.
